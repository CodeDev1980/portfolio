Process of creating

I build websites from front to back, that being the front end to the backend. Being is that I can confirm with clients the layout and content to be displayed on all pages. After it is all good, I build the backend and start replacing dummy content with data that goes through a for loop to iterate data requested or to be sent.

  • 01. Research
  • 02. Design
  • 03. Development

I gather facts about the unique opportunity presented to me and see what works in today's marketing. I work on coding the app so it has fast loading on browsers, holds keywords, and sitemaps that are used for search engines.

I start laying out pages with html, css and JavaScript. Not just making the appearance look great, but also make it function for user activity. Depending on the needs of the website, it may vary from other ones I have built.

After the pages are set and the database is ready, I deploy it to a hosting company that is used by hundreds of other established companies. Because security and performance is also the number concern for any website, I only deploy with proven hosting services.
